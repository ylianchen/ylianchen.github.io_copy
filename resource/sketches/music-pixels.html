Pills

p5.js Generative Art 2022

About this sketch

Pills is a hypnotic, abstract generative animation that creates a continuous stream of dynamic rectangular forms rising from the bottom of the screen. Against a deep blue background, these rectangular shapes—resembling pills or capsules—float upward while rotating and changing scale.

The piece creates an almost meditative experience, with the continuous flow of forms creating patterns that never exactly repeat. Each "pill" has its own properties including size, speed, and slight color variations, contributing to the overall visual complexity while maintaining a cohesive aesthetic.

Visual Elements

The sketch features several key visual elements:

  • Rising Motion: All forms move upward at varying speeds, creating a sense of ascension
  • Rotation: Each shape rotates as it rises, adding dynamic movement
  • Transparency: The semi-transparent nature of the forms allows previous shapes to be partially visible
  • Color Palette: A primarily blue palette creates a cohesive visual field while allowing for subtle variations

Tools Used

  • p5.js for rendering and animation
  • Custom particle system
  • Procedural shape generation

Date

2022

Techniques

  • Procedural motion
  • Color theory application
  • Layered transparency effects
